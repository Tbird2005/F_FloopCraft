Multiplayer

Multiplayer is Peer to Peer. Both players connect directly through the browser using PeerJS. Host creates a room code. The other player types that exact 4 character code. Capital letters matter.
⚠️ VPN warning: VPNs, strict firewalls, school/work networks, and some mobile hotspots can block Peer to Peer connections. Turn off VPNs on both computers if multiplayer will not connect.
